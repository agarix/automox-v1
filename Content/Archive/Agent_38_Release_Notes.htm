Agent 38 Release Notes

Agent 38 is now available to download in the Automox console.

Bug Fixes

  • This release resolves an issue affecting Agent 37 of high CPU utilization during command execution across all devices.

Known Issue

High CPU utilization sporadically affecting macOS devices is root caused and will be resolved in Agent 39.

  • Workaround: If encountered, restarting the Automox agent will restore normal CPU usage
