Agent 40 Release Notes

Release Overview

Agent 1.0.40 is a minor release. This release is for all supported Windows, macOS, and Linux operating systems

Changes / New Features

  • Enablement for PowerShell Signing
    • Automox-authored PowerShell commands will now be signed. This provides the confidence that all Automox-authored PowerShell Scripts are secured and unchanged, from the Automox Cloud to every endpoint.
    • Some known limitations
      • Customer-owned PowerShell commands will not be signed. If they leverage these, then PowerShell commands will remain unsigned
    • PowerShell Signing will be turned on during a phased rollout after they are on Agent 40
  • macOS Installer is now a universal binary
  • Location of supporting files has moved from C:\ProgramData\amagent\ to C:\Program Files\Automox\ for 32-bit, and C:\Program Files (x86)\Automox\ for 64-bit Windows systems. For more detail about file locations, please see Location of Files Required By Automox.

Bug Fixes

  • Security updates
