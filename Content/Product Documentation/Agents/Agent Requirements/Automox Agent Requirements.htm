Automox Agent Requirements

This table lists the system requirements for implementing the Automox agent.


  Windows macOS Linux
Memory utilization 100 MB* 150 MB* 100 MB*
Disk space requirements 70 MB‡ 150 MB‡ 280 MB†‡
OS support For details about the operating systems and versions that Automox supports, see Supported Operating Systems.
PowerShell PowerShell 5.1 or later
  • For agents 1.45.48 and later, the Automox agent uses osqueryi for some portions of its device inventory process. This executable is shipped as part of the agent installer. On Windows systems, osqueryi.exe is used. On other platforms, osqueryi is used.
  • For agents 2.1.44 and later, the Automox Tray introduces a new executable: amagent-ui.exe, a dedicated UI process responsible for the persistent tray experience. This binary is signed on Windows to ensure a trusted and secure experience for both users and IT administrators. This is separate from amagent.exe and not associated with osqueryi.

Notation:

* Varies based on agent activity.
† Varies depending on distro.
‡ Additional free space will be needed to download and install updates.
