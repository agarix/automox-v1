Adding Patches to the Block List in the Automox Console

You can ensure that a patch is blocked from running from the Device Details page or Software page using the Bulk Actions or Actions menu for the patch in question.

On the Device Details page, the list of software and patches are shown at the bottom of the page: 

  1. Select the checkbox for the software you want to block.
  2. Select Ignore from the Bulk Actions drop-down menu.
  3. Confirm the selection.

On the Software page, the Actions drop-down menu can also be used to ignore the patch.

  • Select Ignore from the Actions menu. Note: There is no confirmation message and is effective immediately.
