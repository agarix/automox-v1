Supported Browsers for the Automox Platform

This page describes the supported browsers for the use of Automox.

Note: It’s possible that Automox works for some unsupported browsers. However, for the best experience we recommend using the latest versions of these browsers:
  • Chrome
  • Edge
  • Firefox

Note: With the End of Life for Internet Explorer (IE), Automox no longer supports updates for IE.
