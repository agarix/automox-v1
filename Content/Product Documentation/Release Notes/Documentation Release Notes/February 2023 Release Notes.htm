February 2023 Release Notes

February 24, 2023

Added:

  • New third-party title support:
    • Algodoo (Windows, macOS)
    • Caffeine (macOS)
    • Foxit PDF Editor 11 (Windows)
    • Foxit PDF Editor 12 (Windows)
    • Jabra Direct (Windows)
    • OpenVPN Connect (Win, macOS)
    • OBS Studio (Windows, macOS)
  • New Serial Number column on the Devices page. (Hidden by default)

Updated:

  • Automox Agent Deployer. We updated the executable files for the CrowdStrike agent deployer. Note: The link to this document is new and replaces the previous version.

February 17, 2023

Added:

  • Linux severity scores now support Red Hat devices. The score of No Known CVEs applies to both Ubuntu and RedHat devices. For more details, see Understanding Automox Severity Data.
  • We only support Microsoft 365 apps on Mac devices that were downloaded directly from Microsoft. See Third-Party Software Support.
