August 2024 Release Notes

August 30, 2024

Added:

We added support for these new third-party software titles:

  • IntelliJ IDEA Ultimate 2024.1
  • IntelliJ IDEA Community Edition 2024.1
  • DataGrip 2024.1
  • We no longer use Google’s built-in updater to patch Chrome for Windows
  • Chrome for Windows still patches when running, but this can be overridden so we force-close Chrome before patching.

Updated:

  • Our Roles and Permissions documentation now includes details about API key permissions.

August 23, 2024

Added:

  • The Device Explorer now includes device details directly from the query results table. Select a checkbox to open a sidebar with status details and action options, such as restart, scan, or delete the device. See Device Explorer: Device Details Sidebar.

Updated:

  • We now include naming rules (character length, etc.) for creating a new zone. See Managing Zones: Adding Zones.
  • We consolidated documentation related to deploying the Automox agent using Windows GPO into one document: Deploy the Latest Automox Agent using a Powershell Script via Windows GPO Policy
  • We revised our Agent Firewall Allowlisting Rules. UDP is only recommended for Automox Remote Control. Regarding Ad Blockers: We recommend disabling ad blockers (such as uBlock Origin) for the Automox console, as ad blockers can adversely impact the functionality of the product.

August 16, 2024

Updated:

  • Remote Control Module FAQs
    • Updated TightVNC binary version to 2.8.84
    • Note: For Windows only, when the session ends we remove the TightVNC.msi and TightVNC binary. We reinstall these for every session.

August 2, 2024

Updated:

  • On the Dashboard, the following are not included in the outstanding patch count:
    • Software marked as ignored (set directly in the software table and not through a policy filter)
    • Devices that are excluded from reports
    • See Automox Console Dashboard and Viewing Software Inventory.
  • Updated and improved the previously named Azure AD SAML documentation. See Configuring SAML for Microsoft Entra ID (Azure ID).
