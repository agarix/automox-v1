October 2023 Release Notes

October 27, 2023

Added:

  • New third-party title support:
    • Altova MapForce Professional (Windows) (2020, 2020R2, 2021R2, 2023)
    • Double Commander (macOS)
    • Wondershare UniConverter 14 (Windows, macOS)

Updated:

  • Remote Control FAQs:
    • Automox Remote Control is currently not supported on servers.
    • Updated binaries
    • Browser requirements: Chrome 64, Firefox 79, Edge 79

October 20, 2023

Added:

  • New third-party title support:
    • Adobe Acrobat Reader DC MUI (Windows)
    • Altova MapForce Enterprise (Windows) (2020, 2020R2, 2021R2, 2023)
    • Bitvise SSH Client (Windows)
    • Microsoft ASP.NET Core Runtime 6 (Windows)
    • Oracle Java SE 17, 20 (Windows, macOS)

Updated:

  • Remote Control FAQs:
    • In order for Automox Remote Control to function properly, TightVNC must not be installed on the target device.
    • In addition, the device must be part of a zone that has the Pro package.
  • Agent Firewall Allowlisting Rules: See Additional recommendations for Automox Remote Control.

October 13, 2023

Added:

Updated:

  • Microsoft Office: App is shutdown in order to patch. Refer to our Third-Party Patching Best Practices documentation for details.
  • The Software page now includes filters for Impacted and Days Exposed. See articles related to Software.
  • On the Dashboard, the updated Outstanding Patch Count grid uses the previously listed filters to open the Software Page with the appropriate filters applied. For details, refer to Automox Console Dashboard.
