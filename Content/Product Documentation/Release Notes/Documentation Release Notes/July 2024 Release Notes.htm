July 2024 Release Notes

July 19, 2024

Added:

New third-party title support:

  • Talkdesk Atlas (Windows, macOS)

Updated:

  • Removed deprecated endpoints for old Policy Results Report Service (Developer Portal)

Bugfixes:

  • Devices page: Fixed an issue that incorrectly showed devices as deleted.
  • Profile page: Fixed an issue in which it was not possible to change first and last names.

July 12, 2024

Added:

  • We added a new table to our Agent Firewall Allowlisting article that lists recommended URLs for environments that do not support the use of wildcards. Refer to Recommended Approach Using Specific URLs.
  • We added information about where Automox gets its severity data: Understanding Automox Severity Data.

Bugfixes:

  • Device targeting: The devices preview was not refreshing when the page selector was changed. The number of devices in the preview now reflects the number chosen in the per page option.
  • Remediation Device column was showing a device name as unknown. This is fixed.
  • Software page: Fixed an issue in which the list of impacted devices included ones with the incorrect OS version.
  • Policy Results Report: Fixed an issue in which inaccurate data was being reported.

July 5, 2024

Updated:

  • Zone administrators can now enable and disable Remote Control consent within their environment. See our Remote Control and Roles and Permissions documentation.

Automox Agent 1.44.8 is now officially available. The major feature for this release is AWS Graviton support.

Bugfixes:

  • We fixed an issue in which, for device targeting, the devices preview did not refresh when the devices per page was changed.
