November 2023 Release Notes

November 17, 2023

Added:

  • New third-party title support:
    • Cisco Jabber VDI Client (Windows, macOS)
    • Microsoft Visual Studio Community Edition 2019 & 2022 (Windows)
    • Proton VPN (Windows, macOS)
    • VNC Viewer (Windows, macOS)
  • Script Signing is now available for global and zone administrators. See Automox Script Signing

Updated:

  • Added server group targeting information to Deploying the Automox Agent to Intune-enrolled Devices

November 10, 2023

Added:

  • New third-party title support:
    • Bluebeam OCR (Windows)
    • Bluebeam Revu (Windows)
    • Ekahau AI Pro (Windows, macOS)
    • Epson Easy Interactive Tools (Windows, macOS)
    • Logitech SetPoint (Windows)
    • Wondershare Creative Center (Windows)

Updated:

  • We've updated the Agent Firewall Allowlisting Rules documentation

November 3, 2023

Added:

  • New third-party title support:
    • GoTo Connect (Windows, macOS)
    • Microsoft Azure CLI (Windows)
    • PDF-XChange Editor (Windows)
    • Wondershare UniConverter 13 (Windows, macOS)
  • New Secrets Management:
    • Automox Secrets Storage is a feature designed to enhance the security and convenience of managing sensitive information for IT administrators, particularly when working with Worklet policies in the Automox platform.
  • Added support of macOS 14 (Sonoma). See Supported Operating Systems

Updated:

  • Creating a Worklet:
    • Worklet policy now includes Inputs. See also updated links and images.
  • Using Automox Vulnerability Sync:
    • When opening and saving scanner exports, we recommend not using Notepad++ as it can cause the import to fail.
    • Noted that patchable vulnerabilities include supported third-party software packages.
