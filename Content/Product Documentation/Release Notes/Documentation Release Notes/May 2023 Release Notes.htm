May 2023 Release Notes

May 26, 2023

Updated:

  • We updated the SAML description in our Security article. This includes how to disable two-factor authentication. Images have also been updated.

May 19, 2023

Added:

  • New Helpdesk Operator role. See our Roles and Permissions article.
  • New third-party title support:
    • ActivePresenter 6, 7, and 8 (Windows, macOS)
    • ActivePresenter 9: In addition to Windows, we now support macOS
    • Agent Ransack (Windows)
    • Attendant Pro (Windows)
    • DBeaver Community Edition (Windows, macOS)
    • JAWS 2022 (Windows)

Updated:

  • The console Dashboard now includes a chart that shows the device OS distribution for the current zone.
  • Note: The Last System Scan can periodically be out of sync until the scanning process completes. The timestamp is updated when a scan is initiated and when it ends. See our Device Details article.
  • We now refer to “Restart” in the User Notification section of our policies. These articles reflect this change:
    • Managing End-User Notifications
    • Using Restart Notifications

May 12, 2023

Added:

  • New third-party title support
    • Auto Dark Mode (Windows)
    • AutoIt (Windows)
    • IntelliJ IDEA Ultimate 2023.1 (Windows, macOS)
    • Microsoft Visual C++ Redistributable (2015–2022) (Windows)
    • Zulu JDK 17 (Windows)

Updated:

  • The timestamp for the Last System Scan is updated when a scan is initiated and when it ends. See our Device Details article for more information.

May 5, 2023

Updated:

  • Supported Operating Systems: We’ve added support for these OSes:
    • Fedora 36
    • Ubuntu 22
    • Removal of end of life OSes: CentOS 8, Fedora 34 and Fedora 35
  • Best Practices documentation: These have been updated to show revised UI design and the new Dashboard.
  • The console Dashboard now shows Scheduled Polices organized into tabs for each type of policy. A new link to a list of Devices was added under the Outstanding Patch Count grid.
  • We completed a rebranding of our Developer Portal.
  • We updated our Quick Start Guide - Patch Management in Minutes with a list of character limits for the console.
