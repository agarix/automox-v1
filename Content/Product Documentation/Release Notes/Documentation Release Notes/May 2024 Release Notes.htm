May 2024 Release Notes

May 31, 2024

Updated:

  • We updated the commands for removing the agent for macOS. See Removing the Automox Agent.

New revisions for our third-party supported software documentation:

  • We renamed Zoom Rooms to Zoom Rooms (ZoomPresence on macOS).
  • We renamed Bitwarden and added a section later in the article that details how to install Bitwarden for all users. See Third-Party Software Support.
  • We corrected the Zoom and Docker Desktop values in Third-Party Patching Best Practices.

May 24, 2024

Updated:

  • Create Policy page: We simplified the path to create a policy, which allows you to create policies with fewer clicks. See Managing Policies.
  • Add Device: We added a copy button and adjusted the script within the Add Device modal for macOS agent installation using the download method. See Automox Agent Installation Overview.

Third-Party Software Support:

  • We did a thorough review of our Third-Party Software documentation. Some titles have been adjusted alphabetically. One title was missing the corresponding OS version (DYMO Label is supported for both macOS and Windows).
  • We now provide a more detailed breakdown of the behavior of patching. You can now find out the source of when an app is shut down in order to patch. See Third-Party Patching Best Practices.

Bugfixes:

  • We fixed an issue in which the catalog version for Chrome (macOS) was incorrect.

May 10, 2024

Added:

  • New third-party titles:
    • Amazon Workspaces (Windows, macOS)
    • QZ Tray (Windows)
    • Note: For already existing titles, some cached package names have been broken out in more detail. For example, for the software title DisplayFusion, we now list DisplayFusion_EXE and DisplayFusion_MSI. For the latest, refer to Naming Scheme for Third-Party Software Packages.
  • Device Explorer is a new way to interact with devices within your organization. The Device Explorer was added as a new tab, next to our original device view, that enables you to create searches that are as simple or as complex as you want using the new query composer, while simultaneously viewing a table of results. See Device Explorer.

May 3, 2024

Added:

  • We now officially support Amazon Linux 2023. See our Third-Party Supported Software documentation.

Updated:

  • Automox Agent has been updated. See Agent 1.43.135/1.43.142 Release Notes for details.
  • We recently updated AVR and Vulnerability Sync documentation. This includes updated images and clearer descriptions about using filters. The Devices table layout was updated and the vulnerability import dialog now shows that the import file can have up to 1 million rows. See documentation for Automated Vulnerability Remediation (AVR) and Vulnerability Sync.

Bugfixes:

  • We fixed an issue related to the Splunk app and inaccurate display of data.
  • We fixed an issue with the Policy Name filter in the Activity Log.
  • We fixed an issue related to authenticating with the Secure Token.
