August 11, 2023

Added:

  • New: As part of Automox FixNow™, you can deploy worklets on individual devices and observe real-time feedback through the console. Refer to our Run Worklet Now documentation.

  • New third-party title support:
    • Amazon AWS VPN Client (Windows, macOS)

    • DAX Studio (Windows)
    • Twingate (Windows)

Updated:

  • Refer to Filtering and Searching on the Devices Page to see updates related to the Devices Page:
    • Under the filters, you can now see that Needs Attention is a parent filter with the following children filters available:
      • Disconnected for 30+ Days
      • Not Compatible
      • Needs Reboot
  • The Devices page includes a new filter Show Excluded from Reports, and general improvements when using filters.
