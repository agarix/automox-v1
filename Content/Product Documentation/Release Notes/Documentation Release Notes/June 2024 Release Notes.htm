June 2024 Release Notes

June 28, 2024

Added:

  • Audit Trail API is now available:
    • Audit Trail REST API
    • OCSF Schema Overview
    • OCSF Class Events
  • Third-Party Software: You can now set overrides for how third-party software packages that won't patch while running are handled. To do this, use the new Worklets available in our Worklets Catalog. Refer to Third-Party Patching Overrides
  • To be able to use the worklet for macOS, you will need an identifier for the software packages called Bundle Names. See Naming Scheme for Third-Party Software Packages

Updated:

  • The current search scope for the Device Explorer table is limited to Custom Name, Device Name, and Device UUID. See Device Explorer.

Bugfixes:

  • Fixed an issue in which the Device Explorer was not showing all devices from the Group attribute.
  • Fixed odd behavior related to the UseWUServer reg entry when switching the Automatic Updates settings in the Group settings.
  • Fixed an issue that a report incorrectly showed Chrome was patched.
  • We modified how we show the version of Zoom Outlook Plugin 32-bit.
  • Fixed how we cache the most recent version of Google Chrome for macOS and Windows.

June 21, 2024

Added:

  • CSV Export is now available for the Policy Results Report. See Policy Results Report.

Updated:

  • Third-Party Supported Software: added details about support of PuTTY (Windows). See new section: PuTTY (Windows).
  • Updated Device ID to Device UUID in the Device Explorer device query, table and export file. See Device Explorer.
  • The Device Details > Troubleshooting now includes scan status: successful/failed. See Device Details.

New Beta Agent 1.44.5 Release Notes.

Bugfixes:

  • Fixed issue in which Automated Vulnerability Remediation (AVR) was not linking hosts to their counterparts in Automox.
  • Fixed an issue in which soft deleted zones impacted the entitlement for non-deleted zones.
  • Fixed an issue in which updates were reported to the dashboard, but there were no updates to install.
  • Fixed an issue of missing severity data on the software section of the Device Details page.
  • Fixed an issue on the Device Details page in which the last user login showed a value of null.
  • Fixed pagination issue on Device Explorer page.
  • Fixed issue of agent disconnecting on Linux devices after upgrade.

June 14, 2024

Updated:

Policy Results Report:

  • We completely updated our Policy Results Report, which now provides historical data and includes data from all policy types: patch, required software, and Worklets. This report provides new status information, remediation worklet exit codes, error and log information.

Bugfixes:

  • We fixed an issue related to Device Explorer pagination.
  • We fixed an issue on the Device Details page related to the display of last user login date values.
  • We fixed an issue of missing severity data on the Device Details software section.
