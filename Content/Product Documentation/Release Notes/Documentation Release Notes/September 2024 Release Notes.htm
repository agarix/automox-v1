September 2024 Release Notes

September 27, 2024

Added:

New third-party title support:

  • Microsoft OpenJDK 21 (Windows, macOS)

Updated:

  • Enhanced the Device Explorer to include a new NOT operator in the query builder. See Device Explorer.
  • Updated the TightVNC version for Remote Control from 2.8.84 to 2.8.85 (for both x64 and x86). See Remote Control - FAQs, Known Issues, and Troubleshooting Guide.
  • Updated macOS versions to include Sequoia (macOS 15) in Remote Control - FAQs, Known Issues, and Troubleshooting Guide.

September 20, 2024

Added:

New third-party title support:

  • PyCharm Community Edition (2024.1) (Windows, macOS)
  • PyCharm Professional (2024.1) (Windows, macOS)

Updated:

  • Revised content in Automox Agent Update Policy and added link to Agent Versioning and Support.
  • Added support for macOS 15 (Sequoia) and removed support for macOS 11 (Big Sur). See Supported Operating Systems.

September 13, 2024

Added:

  • We now support Debian 12 and Ubuntu 24. See Supported Operating Systems
  • New: Agent Versioning and Support

Updated:

  • We updated the Location of Files Required By Automox. For Linux and macOS: Agent command logs are written to /var/log/amagent/command.log
  • The command to perform a device compatibility check depends on the agent version. See Agent Command Line Guide.
  • We updated the structure of data returned for the getOrganizations endpoint. See Organization Details in the Developer Portal.

Bugfixes:

  • CVE data was not shown for some RHEL packages. This is now resolved.
  • An issue where device compliance was mismatched between the console and export has been resolved.
  • An issue preventing the policy from being saved when a file > 600MB is uploaded has been resolved.
  • An issue where Function Exclude-WinUpdateApiPreview in GetSoftware was unable to handle a blank UuSSettings.json file has been resolved.
  • An issue where SecureToken is showing as Not Compatible even when the command line returns positive results has been resolved.
