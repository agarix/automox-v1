January 2024 Release Notes

January 26, 2024

Added:

  • New third-party title support:
    • Gpg4win (Windows)
    • Graphmatica (Windows)

Updated:

  • Automox Dashboard: Device Health by Group: We removed the health score and updated the format of time intervals. The default is now set to the Last 7 Days.
  • We updated the Automox Agent Deployer documentation to now require that Falcon scripts are enabled because CrowdStrike updated their console/API.
  • Updated Automox Script Signing: new execution policy requirement for script signing before opting in.

Bugfixes:

  • We fixed an issue with the sort functionality on the Associated Policies table of the Device Details page.
  • We fixed an issue with the display of severities on the Software section of the Device Details page.

January 19, 2024

Added:

  • You can now execute the Run Now command on multiple devices. For more information, see Run Worklet Now
  • New third-party title support:
    • ImgBurn (Windows)
    • VNC Server (Windows)
  • Removed these titles from our third-party software table because they are now handled by Windows Update:
    • Microsoft .NET Desktop Runtime 6
    • Microsoft ASP.NET Core Runtime 6

Updated:

  • Updated Roles and Permissions table:
    • New entry: Automated Vulnerability Remediation (AVR)
    • Updated: Ask Otto permission description
    • Updated Device permission description to Remote Control
  • Updated proxy server documentation for clarity.
  • Updated Okta SSO documentation relative to instructions for attributes and associated image.

Bug Fixes:

  • A bug where groups could not be deleted if deleted devices were still associated with the group has been fixed.

January 12, 2024

Added:

  • New third-party title support:
    • Oracle Java SE 18, 19, 21 (Windows, macOS)
    • GoodSync (Windows)
  • We added new firewall recommendations for Automox Remote Control. For details, see Agent Firewall Allowlisting Rules

Updated:

  • Updated the Troubleshooting section of the Device Details page to help clarify the compatibility check. See Device Details: Troubleshooting

With this release and going forward, we now include information regarding recent relevant bug fixes.

Bug Fixes:

  • We fixed an issue where Adobe Reader 2020 Classic was being overwritten by Adobe Reader DC.
  • We fixed an issue where the serial number column was not shown in the CSV export for the Devices page.

January 5, 2024

Added:

  • New third-party title support:
    • DataGrip 2023.3 (Windows, macOS)
    • GlassWire (Windows)
    • Go Connect (macOS)
    • HandBrake (Windows, macOS)
  • The dashboard now includes a Device Health by Group chart that shows a count of devices by group that have outstanding patches. This also provides a health score, which is an average calculation based on selected groups and the patch age of outstanding patches selected (= devices with no outstanding patches / total devices in selected group). For more information, refer to Automox Console Dashboard
  • Worklet Catalog search functionality has expanded. Searches now query against Worklet Name, description, keywords, and OS. We now support a fuzzy search by Name, and a wildcard match for other metadata terms. Search results are now returned by relevancy rather than by Worklet name. See Using the Worklet Catalog

Updated:

  • Activity Log description: The Automox console retains 90 days of data, which is accessible in increments of 30 days. See Creating Reports
  • Updated the Roles and Permissions matrix for accuracy. Permissions for Devices now refer to Remote Control (instead of Control). Patch Operator permissions have been corrected.
