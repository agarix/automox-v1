Agent 1.42 Release Notes

Release Overview

Agent 1.42.13 is a minor release that addresses multiple issues. This release is for all supported Windows, macOS, and Linux operating systems.

Changes / New Features

Special characters

  • When you enable macOS Secure Token on an M1 device, you can now use the following additional special characters: " ~ % *
    (See Agent 39 Release Notes for password rules.)

Allowlist request

  • To ensure future compatibility of newer components not being blocked in your ecosystem, subscribe to and follow Agent Firewall Allowlisting Rules. This allowlist page will be updated with new components as they are being developed.
