Agent 1.42.22 Release Notes

Release Overview

This release is supported on Windows, macOS, and Linux operating systems. This only contains a fix for macOS and will only auto-update on macOS.

Changes / New Features

Bug fixes

  • Service Account not being enabled for Secure Token with the %!s(<nil>) error
    (See Agent 1.42 Release Notes and Agent 39 Release Notes for password rules)
  • Various security fixes
