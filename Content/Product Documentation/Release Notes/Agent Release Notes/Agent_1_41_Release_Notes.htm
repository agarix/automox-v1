Agent 1.41 Release Notes

Release Overview

Agent 1.41.125 is a minor release that addresses multiple issues. This release is for all supported Windows, macOS, and Linux operating systems. 

Changes / New Features

Naming conventions of Agent Releases

  • We have shifted our old naming convention. Previously, the agent number was associated in the patch section as indicated: 1.0.37, 1.0.38 …. 1.0.41, which is not industry standard
  • We will now be following industry standards for Major, Minor, and Patch

  • We will be providing release notes for all Major and Minor releases

Allowlist request

  • To ensure future compatibility of newer components not being blocked in your ecosystem, subscribe to and follow Agent Firewall Allowlisting Rules. This allowlist page will be updated with new components as they are being developed.

Bug Fixes

  • macOS StompClient fix

This release note has been revised to reflect an updated format of all agent release notes going forward. (January 19, 2023)
