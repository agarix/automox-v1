Agent 2.1.45 Release Notes

Automox Agent version 2.1.45 is a hotfix and includes the following:

Bugfix:

  • Resolved a potential issue in the Automox Tray where if the notification window is zoomed, or the text size enlarged, the button on the bottom of the window could be cut off. This has been resolved in the Automox Agent Version 2.1.45.
This agent version is available via manual update.
