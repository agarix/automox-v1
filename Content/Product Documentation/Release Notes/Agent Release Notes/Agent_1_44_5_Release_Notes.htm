Agent 1.44.5 Release Notes

This version of the Automox Agent is currently in Beta, and includes the following:

  • Support for installation on AWS Graviton instances running Automox-supported Linux distributions
  • Deprecation of the command switch --checkcompat. (Use amagent selftest instead)

Defect Fixes:

  • Agent version 43 not always respecting Proxy information
  • Agent disconnected after upgrade
