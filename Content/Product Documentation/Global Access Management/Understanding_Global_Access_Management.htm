Understanding Global Access Management

Global access management allows a Global Administrator of a company's IT infrastructure to manage all organizations and users from a centralized Global View.

A global administrator has exclusive access to the Global View configuration pages. Organization administrators and other administrator accounts do not have access to the Global View.

Global Administrators have the following permissions:

  • They can view all users for all organizations
  • Add, modify, and remove users
  • Add and modify organizations
  • Configure settings (session timeouts)
  • The Global Administrator has full administrator permissions for all organizations tied to an account

For more information about the global administrator role, refer to Roles and Permissions.
