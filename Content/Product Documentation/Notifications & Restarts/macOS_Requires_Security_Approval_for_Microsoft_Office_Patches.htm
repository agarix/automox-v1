macOS Requires Security Approval for Microsoft Office Patches

Apple has introduced a security measure in macOS Mojave (10.14) and newer that is a requirement for some third-party updates. macOS uses the Microsoft AutoUpdate for patching Microsoft Office applications which is the Microsoft supported mechanism for this OS.

The agent (amagent) now leverages Microsoft AutoUpdate for patching Microsoft Office on devices running on macOS.

  • When macOS prompts you for the security approval for the agent to access Microsoft AutoUpdate, click OK. You only need to do this one time.
